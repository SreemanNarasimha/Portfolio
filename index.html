Digital Safe with Encrypted PIN

A high-security digital safe system powered by STM32.

The user inputs a PIN via a matrix keypad. The microcontroller encrypts this input using the AES algorithm and validates it against the stored hash in the EEPROM. Successful validation triggers a servo motor to unlock the safe, while repeated failures initiate a temporary lockout.

STM32 Embedded C AES Encryption EEPROM
